# ❌⭕ Tic Tac Toe

A 2-player Tic Tac Toe game built with Python.

## 🚀 About The Project
A classic Tic Tac Toe game where two players take turns marking spaces on a 3×3 grid. The player who places three marks in a row wins!

## ✨ Features
- 2 player mode
- Clean board display after every move
- Win detection (rows, columns, diagonals)
- Draw detection
- Play again option

## 🛠️ Built With
- Python 3
